/************************************************* * scan2 — Backend API for Network Asset Scanner * Reliable for GitHub Pages + iPhone/iPad/Android * - GET endpoints: ping, checkDuplicates, submit * - Writes to tabs: Switches, AP, Other * - AssetTag is NEVER required *************************************************/ const SPREADSHEET_ID = "10qscpdita1SHx8lD_AMNusvMe9gKiBIWg-xussXzuxc"; // your sheet id const ACCESS_KEY = "FIT-SCAN-2026"; // your key function requireKey_(key) { if (!key || key !== ACCESS_KEY) throw new Error("Unauthorized — invalid access key."); } function getSS_() { return SpreadsheetApp.openById(SPREADSHEET_ID); } function getSheetByNameLoose_(ss, name) { const target = (name || "").toString().trim().toLowerCase(); return ss.getSheets().find(s => (s.getName() || "").toString().trim().toLowerCase() === target) || null; } function json_(obj) { return ContentService .createTextOutput(JSON.stringify(obj)) .setMimeType(ContentService.MimeType.JSON); } function pick_(v) { return (v === undefined || v === null) ? "" : String(v); } /************************************************* * Duplicate check across Switches/AP/Other * - Asset tag is NOT required and NOT used to block * - We only block if SERIAL duplicates (your preference) *************************************************/ function checkDuplicates_(data) { const ss = getSS_(); const sheetNames = ["Switches", "AP", "Other"]; const serial = pick_(data.serial).trim(); let issues = []; if (!serial) { return { ok: true, issues: [] }; } sheetNames.forEach(name => { const sh = getSheetByNameLoose_(ss, name); if (!sh) return; const values = sh.getDataRange().getValues(); if (!values || values.length < 2) return; values.shift(); // header values.forEach((row, idx) => { // Timestamp | Make | Model | Serial | AssetTag const rSerial = pick_(row[3]).trim(); if (serial && rSerial && serial === rSerial) { issues.push({ field: "serial", message: `Serial already exists in ${name} (row ${idx + 2})` }); } }); }); return { ok: issues.length === 0, issues }; } /************************************************* * Save record * Columns: Timestamp | Make | Model | Serial | AssetTag * AssetTag NEVER required (can be blank) *************************************************/ function submitNetworkAsset_(form) { const ss = getSS_(); const deviceType = pick_(form.deviceType).trim(); // Switch | AP | Other const make = pick_(form.make).trim(); const model = pick_(form.model).trim(); const serial = pick_(form.serial).trim(); // REQUIRED const assetTag = pick_(form.assetTag).trim(); // OPTIONAL if (!deviceType) throw new Error("Device Type required."); if (!make) throw new Error("Make required."); if (!model) throw new Error("Model required."); if (!serial) throw new Error("Serial required."); // (MAC should be stuffed into serial by frontend if no serial) const sheetName = deviceType === "Switch" ? "Switches" : deviceType === "AP" ? "AP" : "Other"; const sh = getSheetByNameLoose_(ss, sheetName); if (!sh) throw new Error(`Sheet "${sheetName}" not found. Check tab names.`); sh.appendRow([new Date(), make, model, serial, assetTag]); return { ok: true, message: "Saved to " + sh.getName() }; } /************************************************* * API entrypoints * * GET /exec?action=ping&k=KEY * GET /exec?action=checkDuplicates&k=KEY&serial=... * GET /exec?action=submit&k=KEY&deviceType=...&make=...&model=...&serial=...&assetTag=... *************************************************/ function doGet(e) { try { const p = (e && e.parameter) ? e.parameter : {}; const action = pick_(p.action).trim(); requireKey_(pick_(p.k)); if (action === "ping") { return json_({ ok: true, message: "pong" }); } if (action === "checkDuplicates") { const result = checkDuplicates_({ serial: p.serial }); return json_({ ok: true, result }); } if (action === "submit") { // optional: enforce duplicate block here too const dup = checkDuplicates_({ serial: p.serial }); if (!dup.ok) return json_({ ok:false, error: dup.issues.map(x=>x.message).join(" | "), issues: dup.issues }); const result = submitNetworkAsset_(p); return json_(result); } return json_({ ok: false, error: "Unknown action" }); } catch (err) { return json_({ ok: false, error: (err && err.message) ? err.message : String(err) }); } }
