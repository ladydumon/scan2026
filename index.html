Network Asset Scan

Default mode: Take a photo (native camera autofocus) → OCR reads Serial + Model/Make. Serial is preferred. If no Serial is found but MAC is found, MAC will be placed into Serial.

Asset Tag is optional and never required.
Motorola TW-511 is treated as Switch.
If OCR finds MAC but not Serial, it will put MAC into this Serial field.
Tip: fill the label area in the frame. If text is tiny, get closer and keep it flat (no glare).
